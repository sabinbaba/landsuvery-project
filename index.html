Capstone Project · Abstract

About This Project

A final-year Geomatics Technology capstone by BYIRINGIRO Jean Bosco (Reg. 25RP18520), supervised by MEng. MUTIMUCYEYE Marie Grace — Academic Year 2025–2026.

The rapid expansion of campus facilities at RP Ngoma College — including academic buildings, administrative blocks, roads, water supply systems, electricity networks, drainage lines, and communication infrastructure — has increased the need for an accurate, up-to-date, and easily accessible spatial information system.

Currently, much of the infrastructure information is managed through manual records and paper-based maps, which are prone to inaccuracies, data loss, and inefficiency in maintenance planning and decision-making.

This project develops a comprehensive digital campus map using Geographic Information Systems (GIS) integrated with Digital Global Positioning System (DGPS) technology to collect, analyze, store, and visualize spatial and attribute data related to campus infrastructure and utilities. The resulting digital map supports efficient infrastructure management, maintenance scheduling, emergency response planning, resource allocation, and future campus planning — serving as a model for other educational institutions seeking "smart campus" status.

Develop a comprehensive digital campus map using GIS integrated with DGPS technology to collect, analyze, store, and visualize spatial and attribute data related to campus infrastructure and utilities at RP Ngoma College.

Specific Objective 1

Chapter 3 · Methodology

Data Acquisition & Workflow

A multi-phase approach combining primary DGPS field surveying with secondary data digitization, processed in ArcGIS 10.7 to produce the final thematic map outputs.

01
Planning & Preparation
Define all required data layers and attribute structures in the GIS project. Prepare field data collection forms. Conduct a preliminary reconnaissance to understand campus layout and identify data collection challenges.
02
Primary Data Collection — DGPS Field Survey
Systematic walk-over survey of the entire campus using high-precision DGPS receivers to collect coordinates (x, y, z). Features captured: Point features (manholes, water valves, electric poles, fire hydrants), Line features (road centrelines, utility routes, drainage channels), Polygon features (building footprints, parking lots, sports fields).
03
Secondary Data Collection — Document Digitization
Gathering existing campus site plans, architectural drawings, and utility schematics from RP Ngoma College administration. Open-source satellite imagery (Google Earth, Sentinel Hub) geo-referenced and used as a base for on-screen digitizing in GIS software.
04
Data Integration & Georeferencing
Import DGPS data and digitized secondary data into ArcGIS 10.7. Register raster images using Ground Control Points (GCPs) — a process known as rubber sheeting or georeferencing. Coordinate system: WGS 84 / UTM Zone 35S (EPSG:32735) for national consistency.
05
GIS Database Development
Create a spatial database with different layers for each infrastructure category. Link attribute data to spatial features (Building_ID, Name, Type, Year_Built, Floors, Department; Utility_ID, Type, Material, Diameter, Depth, Installation_Date). Store as GeoPackage or Shapefile format.
06
Data Verification & Quality Assurance
Cross-check field data with existing plans to resolve discrepancies. Editing and verification addresses: incompleteness of spatial data, locational placement errors, distortion, and incorrect linkages between spatial and attribute data.
07
Digital Map Production & Spatial Analysis
Produce 2D planimetric and 3D perspective maps with standard cartographic elements. Apply overlay analysis, buffer analysis (distances between utilities and buildings), and network analysis (road/pathway connectivity). Symbolize layers with appropriate colors, patterns, and legends.
